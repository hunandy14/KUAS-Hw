#### 作業內容：
本次作業內容主要是熟悉樣板的基本使用方法。   
請設計一個小程式，能夠計算陣列裡的元素之  

1. 總和、
2. 總乘積還有
3. 總平方之和。  

例如：element[3] = { 5, -2, 4 }   

<pre>Total sum = 7 (5 – 2 + 4)
Total product = -40 (5 * (-2) * 4)
Total square of sum = 45 (5*5 + (-2)*(-2) + 4*4)   </pre>

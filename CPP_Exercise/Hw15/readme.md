#### 作業內容： 
本次作業內容主要是熟悉 C++裡的字串使用方法(非傳統字串用法)。  
請設計一個小程式，2 種字串處理功能  

1. 輸入一數字字串改成英文字串輸出  
2. 輸入一字串依每 N 位元分開  
3. 輸入一字串(包含空白)，並將每個英文字的字母數放在英文字之後  


#### 例如：

1. String  = `1245`  
	Output => `OneTwoFourFive`
2. String  = `aaabbbccc`   >2 位元  
    Output => `aa ab bb cc c`
3. String  = `A apple a day`  
    Output => `A[1] apple[5] a[1] day[3]`

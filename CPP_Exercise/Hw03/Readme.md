#### 作業內容：  
本次作業內容主要是熟悉型別與算子的使用方法。  
請用程式設計寫出以下的例子，輸入兩個浮點數 X、Y，  
第一個浮點數 X 是要使用的值，第二個浮點 Y 數代表精確度，  
例如 X=0.1234、Y=0.01，輸出 0.1200。   

#### 作業要求：
1. 直接為 0 (例如 X=0.5555、Y=0.01，輸出 0.5500)  
2. 直接進位 (例如 X=0.1234、Y=0.01，輸出 0.1300)  
3. 四捨五入 (例如 X=0.5555、Y=0.01，輸出 0.5600)  
4. 精確度 Y 分別要有 1、0.1、0.01、0.001
